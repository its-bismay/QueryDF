üîå API Endpoints

POST /api/upload
Upload a PDF document for processing and indexing.
Request:
POST /api/upload Content-Type: multipart/form-data pdf: [PDF File]
Response:
{ "success": true, "url": "https://res.cloudinary.com/...", "pdf_name": "teams_todo_assignment" }
POST /api/query
Ask questions about an uploaded PDF document.
Request:
POST /api/query Content-Type: application/json { "query": "What is this PDF about?", "pdfName": "teams_todo_assignment" }
Response:
{ "answer": "This PDF is about an assignment to build...", "sources": [ { "sourceId": 1, "chunkIndex": 0, "source": "https://res.cloudinary.com/...", "score": 0.22112079, "text": "Teams ToDo Build a collaborative..." } ] }
